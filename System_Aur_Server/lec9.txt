Topic: Linux User Management & Linking:-


* User & Group Commands:-

- useradd [username]: Naya user banane ke liye.

- groupadd [groupname]: Naya group banane ke liye.

- cat /etc/passwd: Saare users check karne ke liye.

- cat /etc/group: Saare groups check karne ke liye.
 
- gpasswd -a [user] [group]: Ek user ko group mein add karne ke liye.



* Linking in Linux:-

- Soft Link (Symbolic Link): Ye ek shortcut ki tarah hota hai. Command: ln -s [source] [link_                             name].

- Hard Link: Ye file ka backup/mirror hota hai. Command: ln [source] [link_name].



* Archiving & Downloading:

- tar -cvf: Files ko bundle (tarball) banane ke liye.

- gzip: Files ko compress karne ke liye.

- wget [URL]: Internet se direct files download karne ke liye.
