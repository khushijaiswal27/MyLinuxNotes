Topic: Linux User Management & Linking:-

* User & Group Commands:-

- useradd [username]: Command to create a new user.

- groupadd [groupname]: Command to create a new group.

- cat /etc/passwd: Command to check/list all users in the system.

- cat /etc/group: Command to check/list all groups in the system.

- gpasswd -a [user] [group]: Command to add a specific user to a group.

* Linking in Linux:-

- Soft Link (Symbolic Link): Functions like a shortcut in Windows. If the original file is deleted, the link breaks.

- Command: ln -s [source] [link_name]

- Hard Link: Functions as a mirror or a backup of the file. Even if the original file is deleted, the data remains accessible through the hardlink.

- command: ln [source] [link_name]

* Archiving & Downloading:-

- tar -cvf: Used to bundle multiple files into a single archive (tarball).

- gzip: Used to compress files to reduce their size.

- wget [URL]: Used to download files directly from the internet via the terminal.
