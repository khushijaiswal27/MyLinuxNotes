--- Lecture 8: Package Management & System Info ---

1. System Pehchaan (System Info)

- hostname: System ka naam dekhne ke liye.
- hostname -i: System ki IP address janne ke liye.
- cat /etc/os-release: OS ka version aur details dekhne ke liye.

2. Software Lagana (Package Management - YUM) :
 
- sudo yum install httpd -y: Apache Web Server install karne ke liye.
- sudo yum update -y: Saare softwares ko update karne ke liye.
- sudo yum remove [package]: Software hatane ke liye.

         install ya systemctl start likh sakti hain .Like this:-

- yum install httpd -y: Apache Web Server install karne ke liye.
- yum update -y: Saare softwares ko update karne ke liye.

3. Service Control (Server Management):-

-  systemctl start httpd: Server chalu karne ke liye.
-  systemctl status httpd: Dekhne ke liye ki server "Active" hai ya nahi.
-  systemctl stop httpd: Server band karne ke liye.
-  chkconfig httpd off:  ab system ko manually har bar start krna pdenga.
-  chkconfig httpd on: Reboot ke baad bhi server apne aap chalu ho jaye.
                               OR
-  systemctl enable httpd : (purana tarika  chkconfig hai , par naya version ye hai) 
                           Isse server restart ke baad bhi apne aap start ho jata hai.
