1. Basic Introduction:-

* pwd (Print Working Directory): Displays the full path of the current directory you are in.

* whoami: Displays the username of the current user (e.g., ec2-user).

* clear: Clears the terminal screen for a better view.

2. Navigation:-

* ls (List): Lists the files and folders inside the current directory.

* cd [folder_name] (Change Directory): Used to enter a specific folder.

* cd ..: Moves one level back to the parent directory (Note: space is mandatory).

* cd ~: Instantly returns to the user's home directory.

* cd -: Toggles back to the previous working directory.

3. File Management:-

* touch [file_name]: Creates an empty file.

* cat [file_name]: Displays the content of a file directly in the terminal.

* mv [source] [destination]: Used to move a file or rename it.

* cp [source] [destination]: Creates a duplicate copy of a file.

4. Hidden Files:-

* How to create a hidden file: Add a dot (.) at the beginning of the filename (Example: .secret.txt).

* How to view hidden files: Use the command ls -a (All).

5. Path (Absolute vs. Relative):-

* Absolute Path: The full path starting from the root directory (/).

* Relative Path: A path based on your current location in the file system.

6. Vi/Vim Editor:-

* vi [file_name]: Opens the file in the Vi editor.

* Press i: Enters Insert Mode (allows you to start typing).

* Press Esc: Exits Insert Mode to enter Command Mode.

* Type :wq: Saves the changes and exits the editor.
